void esDMemInit ( esDMemHandle_T *  handle,
void *  storage,
size_t  storageSize 
)

Inicijalizuje dinamican memorijski alokator.

Parameters
[out]handleDeskriptor dinamickog alokatora
[in]storagePredefinisani memorijski prostor koji se predaje dinamickom alokatoru na koriscenje
storageSizeVelicina memorijskog prostora u bajtovima

Ova funkcija se mora pozvati pre koriscenja funkcija dinamickog memorijskog alokatora.

Warning
Funkcija zahteva da pokazivaci handle i pool budu poravnani (aligned). Ukoliko se koriste eSolid alokatori za instaciranje handle strukture i poolStorage onda je poravnani pristup osiguran.
Funkcija zahteva da velicina memorijskog prostora storageSize bude poravnana (aligned). Na primer za 32-bitni procesor (poravnanje 4 bajta): ako je storageSize == 313 onda je potrebno poravnati na sledecu vecu vrednost koja je deljiva sa 4, u ovom slucaju ce to biti 316.

Regular API object, this object is part of the application programming interface.
void* esDMemAllocI ( esDMemHandle_T *  handle,
size_t  size 
)

Dodeljuje memorijski prostor velicine size.

Parameters
[in]handleDeskriptor dinamickog alokatora
sizeVelicina zahtevanog memorijskog prostora u bajtovima.
Returns
Pokazivac na rezervisani memorijski blok.

U debug rezimu ova funkcija uvek vraca pokazivac, odnosno, ne moze se desiti da vrati NULL pokazivac, kao sto nalaze standardna implementacija malloc C funkcije. Ukoliko se zahtevana memorija ne moze dobaviti generisace se ASSERT greska. Kada se ne koristi debug rezim funkcija se ponasa u skladu sa standardom.

Function class:
